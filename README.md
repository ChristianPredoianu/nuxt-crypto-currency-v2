<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ChristianPredoianu/nuxt-crypto-currency-v2.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ``` 
3. cd into folder
   ```sh
   cd nuxt-crypto
   ``` 
4. cd into folder
   ```sh
   cd nuxt-crypto
   ``` 
5. serve with hot reload at localhost:3000
   ```sh
    npm run dev
   ``` 
6. build for production and launch server
   ```sh
    npm run build
    npm run start
   ```
7. generate static project
   ```sh
    npm run generate
   ```
